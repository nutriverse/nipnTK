# Example dataset for digit preference function

Anthropometric data from a single state from a Demographic and Health
Survey (DHS) of a West African country.

## Usage

``` r
dp.ex01
```

## Format

A data frame with 796 observations and 6 variables

|              |                               |
|--------------|-------------------------------|
| **Variable** | **Description**               |
| *psu*        | Primary sampling unit         |
| *age*        | Age (months)                  |
| *sex*        | Gender                        |
| *wt*         | Weight (kg)                   |
| *ht*         | height (cm)                   |
| *oedema*     | Presence or absence of oedema |
