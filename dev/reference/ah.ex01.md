# Example dataset for age heaping function

Anthropometric data from a Rapid Assessment Method for Older People
(RAM-OP) survey in the Dadaab refugee camp in Garissa, Kenya. This is a
survey of people aged sixty years and older.

## Usage

``` r
ah.ex01
```

## Format

A data frame with 593 observations and 10 variables

|              |                                  |
|--------------|----------------------------------|
| **Variable** | **Description**                  |
| *psu*        | Primary sampling unit            |
| *camp*       | Camp name code                   |
| *block*      | Block code                       |
| *age*        | Age (years)                      |
| *sex*        | Sex                              |
| *weight*     | Weight (kg)                      |
| *height*     | Height (cm)                      |
| *demispan*   | Demispan (cm)                    |
| *muac*       | Mid-upper arm circumference (cm) |
| *oedema*     | Oedema                           |

## Source

Data courtesy of [HelpAge International](https://www.helpage.org)
