# Example dataset for digit preference

Anthropometric data for a sample of children living in a refugee camp in
a West African country.

## Usage

``` r
dp.ex03
```

## Format

A data frame with 374 observations and 6 variables

|              |                                  |
|--------------|----------------------------------|
| **Variable** | **Description**                  |
| *age*        | Age (months)                     |
| *sex*        | Gender (1 = Male / 2 = Female)   |
| *weight*     | Weight (kg)                      |
| *height*     | Height (cm)                      |
| *muac*       | Mid-upper arm circumference (mm) |
| *oedema*     | Presence or absence of oedema    |
